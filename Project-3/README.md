# SQL Data Analysis using SQLite

## Project Overview

This project focuses on performing SQL-based data analysis on an e-commerce dataset using SQLite and DB Browser for SQLite. The objective is to extract meaningful business insights by applying SQL queries and aggregation functions.

The dataset contains customer orders, product information, payment methods, order status, coupon usage, referral sources, and revenue details.

---

## Objectives

* Import and manage data in SQLite.
* Perform data exploration using SQL.
* Analyze revenue and sales performance.
* Understand customer purchasing behavior.
* Generate business insights using SQL queries and aggregate functions.

---

## Dataset Description

The dataset contains the following attributes:

| Column Name     | Description                 |
| --------------- | --------------------------- |
| OrderID         | Unique order identifier     |
| Date            | Order date                  |
| CustomerID      | Unique customer identifier  |
| Product         | Product purchased           |
| Quantity        | Quantity purchased          |
| UnitPrice       | Price per unit              |
| ShippingAddress | Customer shipping address   |
| PaymentMethod   | Payment method used         |
| OrderStatus     | Status of order             |
| TotalPrice      | Total order value           |
| TrackingNumber  | Shipment tracking number    |
| ItemsInCart     | Number of items in cart     |
| CouponCode      | Applied coupon code         |
| ReferralSource  | Customer acquisition source |

## SQL Concepts Used

The following SQL keywords and functions were used during the analysis:

* SELECT
* FROM
* WHERE
* GROUP BY
* ORDER BY
* COUNT()
* SUM()
* AVG()
* MAX()
* MIN()
* LIMIT

These SQL concepts were applied to perform data retrieval, filtering, sorting, grouping, aggregation, and business analysis.

## Key Findings

* Total revenue generated from all orders was ₹1,264,761.96.
* Product-wise analysis identified the highest revenue-generating products.
* Order status analysis revealed the distribution of delivered, cancelled, pending, returned, and shipped orders.
* Payment method analysis showed customer payment preferences.
* Referral source analysis identified the most effective customer acquisition channels.
* Coupon analysis highlighted the most frequently used promotional codes.

---

## Business Insights

1. High-performing products contribute significantly to overall revenue.
2. Customer payment preferences can help optimize payment options.
3. Effective referral channels can be prioritized for future marketing campaigns.
4. Frequently used coupons indicate successful promotional strategies.
5. Monitoring order status trends can improve operational efficiency and customer satisfaction.
