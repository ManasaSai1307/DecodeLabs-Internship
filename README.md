# 🚀 DecodeLabs Data Analytics Internship

## 📌 Overview

This repository contains the projects completed during my **Data Analytics Internship at DecodeLabs**. Throughout the internship, I worked on real-world datasets and applied data analytics techniques to clean, analyze, visualize, and extract meaningful insights from data.

# 📂 Projects

## 📊 Project 1 – Data Preparation and Processing

### Project Objective

The goal of this project was to prepare raw data for analysis by ensuring data quality, consistency, and accuracy.

### Tasks Performed

* Identified and handled missing values.
* Removed duplicate records.
* Standardized data formats.
* Corrected inconsistencies in categorical and numerical data.
* Validated data quality using Pivot Tables.
* Created charts to summarize key business metrics.

### Key Outcomes

* Produced a clean and reliable dataset.
* Improved data quality and usability.
* Established a strong foundation for further analysis.

### Skills Applied

* Data Cleaning
* Data Validation
* Data Transformation
* Excel Analytics

## 📈 Project 2 – Exploratory Data Analysis (EDA)

### Project Objective

The objective of this project was to explore the cleaned dataset, identify patterns, detect outliers, analyze distributions, and uncover meaningful business insights.

### Tasks Performed

* Performed descriptive statistical analysis.
* Calculated measures such as mean, median, minimum, maximum, and count.
* Conducted distribution analysis using histograms.
* Detected outliers using box plots.
* Performed correlation analysis using heatmaps.
* Explored relationships between variables using scatter plots.
* Analyzed revenue, products, payment methods, referral sources, and coupon usage.

### Visualizations Created

* Histogram
* Box Plot
* Correlation Heatmap
* Scatter Plot

### Key Outcomes

* Identified important trends and patterns.
* Detected anomalies and outliers in the dataset.
* Generated actionable business insights.
* Improved understanding of customer purchasing behavior.

### Skills Applied

* Exploratory Data Analysis
* Statistical Analysis
* Data Visualization
* Business Insight Generation
* Python Programming
