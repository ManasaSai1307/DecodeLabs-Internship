# Data Visualization – E-Commerce Sales Performance Dashboard

## 📌 Project Overview

This project focuses on creating an interactive **E-Commerce Sales Performance Dashboard** using **Power BI**. The dashboard transforms raw sales data into meaningful insights through KPI cards, charts, and visual analytics, enabling better business decision-making.

## 🎯 Objectives

* Analyze overall sales performance.
* Track total revenue, orders, customers, and average order value.
* Understand customer payment preferences.
* Monitor product-wise revenue contribution.
* Analyze order status distribution.
* Identify top referral sources.
* Visualize monthly revenue trends.

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Microsoft Excel** (Data Source)
* **Data Visualization**
* **Business Intelligence (BI)**

## 📊 Dashboard Features

### KPI Cards

* **Total Orders:** 1,200
* **Total Revenue:** ₹12,64,762
* **Total Customers:** 1,189
* **Average Order Value:** ₹1,054

### Visualizations

1. **Revenue by Product**

   * Horizontal bar chart showing revenue generated by each product.

2. **Payment Method Distribution**

   * Pie chart displaying customer payment preferences.

3. **Order Status Distribution**

   * Column chart showing counts of Completed, Returned, Pending, Shipped, and Cancelled orders.

4. **Revenue by Referral Source**

   * Bar chart comparing revenue generated through different marketing channels.

5. **Monthly Revenue Trend**

   * Line chart illustrating revenue fluctuations across months.

## 📈 Key Insights

* Generated **₹12.64 Lakhs** in total revenue.
* Processed **1,200 customer orders**.
* Served **1,189 customers**.
* Average order value was **₹1,054**.
* Revenue contribution was fairly balanced across products.
* Payment methods were evenly distributed, indicating diverse customer preferences.
* Referral sources contributed similar revenue levels, with slight variations.
* Revenue peaked during the middle months of the year before declining.
