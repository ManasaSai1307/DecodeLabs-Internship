📊 Project 2 - Exploratory Data Analysis (EDA) Using Python

📌 Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned e-commerce dataset using Python. The goal was to understand the dataset structure, analyze customer and sales-related information, identify patterns, and derive meaningful business insights through statistical analysis and data visualization.

🎯 Project Objectives

* Understand the dataset structure and contents.
* Analyze data quality and consistency.
* Identify missing values and duplicate records.
* Generate descriptive statistical summaries.
* Explore sales, revenue, and customer-related trends.
* Visualize relationships and patterns within the data.

🛠️ Tools & Technologies Used

* 🐍 Python
* 🐼 Pandas
* 🔢 NumPy
* 📈 Matplotlib
* 🎨 Seaborn

🔍 Exploratory Data Analysis Performed

📋 Dataset Inspection

* Examined dataset dimensions and structure.
* Reviewed column names and data types.
* Displayed sample records for initial understanding.

🧹 Data Quality Assessment

* Checked for missing values.
* Identified duplicate records.
* Validated data consistency.

📊 Statistical Analysis

* Generated descriptive statistics.
* Calculated Mean, Median, Minimum, and Maximum values.
* Analyzed numerical attributes and distributions.

💰 Business Analysis

* Product-wise Revenue Analysis
* Order Status Distribution Analysis
* Payment Method Usage Analysis
* Referral Source Performance Analysis
* Coupon Code Utilization Analysis

📈 Data Visualization

The following visualizations were created:

* Histogram for Total Price Distribution
* Box Plot for Outlier Detection in Unit Price
* Correlation Heatmap for Numerical Features
* Scatter Plot showing Quantity vs Total Price Relationship

💡 Key Insights

* Evaluated revenue contribution across different products.
* Analyzed customer purchasing patterns.
* Studied order status distribution and transaction behavior.
* Identified the most frequently used payment methods.
* Examined referral source effectiveness.
* Investigated coupon usage trends.
* Explored relationships between quantity purchased and total revenue.
